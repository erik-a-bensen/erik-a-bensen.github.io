`tmg-hmc` is a Python implementation of the Exact HMC sampling algorithm from <a href="https://doi.org/10.1080/10618600.2013.788448" target="_blank" rel="noopener" class="project-link">Pacman and Paninski (2014)</a> for truncated multivariate gaussian distributions with linear and quadratic constraints.
